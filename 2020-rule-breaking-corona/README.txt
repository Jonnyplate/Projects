# 2020-rule-breaking-corona

### Files

`index.html`: main experimental file

### Folders and their content
`ExperimentFiles`:

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; `setup.js`: This file declares and initializes constants and basic variables and participant set-up.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; `functions.js`: Contains some additional functions used in the experiment.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; `instructions.js`: Contains the text slides as JSPsych variables.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; `survey_questions.js`: Contains all form- and survey elements (+ scales) displayed in the experiment. 

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; &nbsp;&nbsp;&nbsp; `trials.js`: Defines the different trial and block components.

`JSPsych-master`: contains the latest version of JSPsych.

`jspsych-rule-violation.js`: custom plugin for the paradigm
